You are a senior software architect and accessibility engineer.

Design and implement a **confidence-based grammar and naturalness correction system** for an AAC application. The system should:

1. Take **concept tokens** and generate a deterministic sentence using:

   * Grammar graph validation
   * Morphology engine
   * Prediction engine (n-gram / probability-based)

2. Compute a **confidence score (0–1)** for the generated sentence based on:

   * **Grammar score**: 1 if sentence follows grammar graph rules, <1 if minor deviations

   * **Morphology score**: 1 if verb conjugations, gender agreement, prepositions are correct, <1 if minor issues

   * **Prediction score**: probability of the concept sequence according to the prediction engine

   * Combine scores into overall confidence:

     ```
     confidence = w1*grammar_score + w2*morphology_score + w3*prediction_score
     ```

   * Weights can be tuned; default: `w1=0.4`, `w2=0.4`, `w3=0.2`

3. **Use AI correction only if confidence < threshold**

   * Threshold default: 0.8
   * If confidence ≥ threshold → return sentence as-is
   * If confidence < threshold → call LLM to correct grammar and naturalness

4. **Implement caching**

   * Store every sentence that has been AI-corrected along with input concept tokens and output sentence
   * Before sending a new sentence to AI:

     * Check if the **same concept token sequence** exists in cache
     * If yes → return cached corrected sentence
     * If no → send to AI, store result in cache

5. **System requirements**

   * Must support multiple languages (en, fr, es, pt, it)
   * Sentence length: 3–6 words
   * Output: corrected natural sentence
   * Fallback: if AI fails, return deterministic sentence
   * Cache should persist between app sessions

6. **Expected input/output example**

```json
{
  "concept_tokens": ["I", "WANT", "DRINK", "WATER"],
  "language": "it"
}
```

* Deterministic sentence: `"Voglio acqua"`
* Confidence: `0.95` → return as-is

```json
{
  "concept_tokens": ["I", "GO", "HOME"],
  "language": "fr"
}
```

* Deterministic sentence: `"Je vais maison"`
* Confidence: `0.65` → AI correction → `"Je vais à la maison"`
* Store in cache

---

7. **Implementation suggestions**

   * TypeScript or Python
   * Use lightweight JSON cache (e.g., key = joined concept tokens + language)
   * Modular design:

     ```
     Concept tokens → Grammar/Morphology → Confidence → [Cache → AI] → Sentence
     ```
   * AI correction should only be triggered for **low-confidence sentences**
